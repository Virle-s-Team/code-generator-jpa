# 使用说明

## 1. 基本使用步骤

1. 在实体类上添加@AutoEntity注解
```java
@AutoEntity(
    basePath = "/api/user",
    generateFileType = {GenerateFileType.Controller, GenerateFileType.Service},
    controllerExclude = {ControllerExclude.deleteById}
)
public class User {
    // 实体类字段定义
}
```

2. 配置生成路径(application.yml)
```yaml
code-generator:
  entity-package: com.example.entity
  controller-package: com.example.controller
  service-package: com.example.service
  repository-package: com.example.repository
```

3. 编译项目，自动生成代码

## 2. 注解参数说明

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| basePath | String | 是 | Controller的基础路径 |
| generateFileType | GenerateFileType[] | 否 | 指定生成的文件类型，默认生成全部 |
| controllerExclude | ControllerExclude[] | 否 | 指定要排除的Controller方法 |

## 3. 生成结果

生成的代码结构示例：
```
com.example
├── controller
│   └── UserController.java
├── service
│   ├── UserService.java
│   └── impl
│       └── UserServiceImpl.java
└── repository
    └── UserRepository.java
```

## 4. 注意事项
- 实体类必须使用JPA注解
- 编译时需要启用注解处理
- 生成的代码会覆盖同名文件
